# Sales Performance Dashboard

## Tools Used
Python | SQL | PostgreSQL | Power BI | DAX | Power Query

## Dataset
Superstore Sales Dataset — 9,994 records

## Project Steps
- Collected and cleaned data using Python and Pandas
- Performed EDA to find trends and patterns
- Wrote SQL queries in PostgreSQL using window functions and CTEs
- Built 3 page interactive Power BI Dashboard with DAX measures

## Dashboard Pages
- Page 1 — Sales Overview
- Page 2 — Product Analysis
- Page 3 — Regional Analysis

## Key Insights
- Technology has highest profit margin
- West region Furniture has pricing anomaly
- Q4 has highest sales every year
- Canon imageCLASS is top selling product
